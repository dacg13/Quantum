⌨️
Keyboard Navigation

All controls reachable via Tab. Sliders operable with arrow keys. Gate buttons accessible via Enter/Space.

🔍
Screen Readers

ARIA labels on all interactive elements. Live regions announce state changes. Mathematical formulas use MathML.

🎨
Color Blind Mode

Alternate palette using shape + pattern cues. High contrast mode toggle. Never rely on color alone for meaning.
